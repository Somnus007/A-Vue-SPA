# A-Vue-SPA

一个基于vue框架开发的webapp

项目名称：西十区票务

项目介绍：该项目是一个电影及演出售票类的B2C平台webapp

涉及的技术栈： vue / vue-router / vuex / webpack / sass / iconfont / vue-swipe/ vue dev tools / rem&flex布局 / git / iscroll

功能模块：首页、搜索功能、产品列表、分类切换、详情页、下拉刷新
